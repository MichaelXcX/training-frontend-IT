Block

This is a block-level element. It takes up the full width available and starts on a new line.

Inline

This is an inline-level element. It only takes up as much width as necessary and does not start on a new line.

Inline-block

This is an inline-block element. It behaves like an inline element but allows for setting width, height, padding, and margins.
